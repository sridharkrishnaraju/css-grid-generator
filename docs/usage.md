# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="grid-generator.js"></script>
<grid-generator></grid-generator>
```

## React

```jsx
import "@sgbp/grid-generator";
export default function Page() { return <grid-generator />; }
```

## Vue

```vue
<script setup>
import "@sgbp/grid-generator";
</script>

<template>
  <grid-generator />
</template>
```
